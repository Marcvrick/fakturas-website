e-Kuatia vs e-Kuatia'i: cual sistema de facturacion electronica necesitas?

La DNIT ofrece dos caminos para emitir facturas electronicas en Paraguay: e-Kuatia y e-Kuatia'i. Suenan parecido, pero estan disenados para perfiles de contribuyente muy distintos. Elegir el incorrecto puede significar limitaciones operativas o costos innecesarios. Esta guia te ayuda a decidir cual necesitas.

Dos nombres, dos sistemas, dos perfiles

Antes de comparar funcionalidades, es importante entender que e-Kuatia y e-Kuatia'i no son versiones del mismo producto. Son dos plataformas distintas creadas por la DNIT para resolver necesidades diferentes dentro del Sistema Integrado de Facturacion Electronica Nacional (SIFEN).

En guarani

Kuatia significa "documento" o "papel". e-Kuatia = documento electronico. e-Kuatia'i = documento electronico pequeno (el sufijo 'i en guarani indica algo pequeno o sencillo). Los nombres ya anticipan para quien esta disenado cada uno.

e-Kuatia'i es la plataforma gratuita de la DNIT pensada para pequenos contribuyentes. Funciona directamente desde el portal web de la DNIT, sin necesidad de instalar software ni contratar un proveedor externo. Es el punto de entrada mas simple al SIFEN.

e-Kuatia es el sistema completo de facturacion electronica. Requiere un software certificado por la DNIT que se conecta al SIFEN via API. Es la opcion para negocios con multiples establecimientos, alto volumen de documentos, o necesidad de integracion con sistemas contables existentes.

Quien califica para e-Kuatia'i

e-Kuatia'i no esta disponible para todos. La DNIT establece requisitos especificos para poder usar esta plataforma simplificada:

  • Un unico establecimiento: si tenes mas de una sucursal, local o punto de venta, no calificas. Un solo RUC con un solo local.
  • Un unico punto de emision: solo una terminal o puesto desde donde se emiten las facturas. Si tenes dos cajas registradoras emitiendo facturas, necesitas e-Kuatia.
  • Bajo volumen de facturacion: e-Kuatia'i esta pensado para negocios con un volumen de operaciones que no supere los parametros definidos por la DNIT. Estos parametros pueden variar, por lo que conviene consultar la normativa vigente.
  • Tipos de documento limitados: e-Kuatia'i permite emitir facturas electronicas y boletas electronicas. Si necesitas emitir notas de credito electronicas, notas de remision, o autofacturas, necesitas e-Kuatia.

En la practica: e-Kuatia'i funciona bien para kioscos, pequenos comercios, profesionales independientes y microemprendedores con un solo local y pocas facturas al mes. Si tu negocio tiene mas de un empleado emitiendo facturas o necesitas automatizacion, probablemente ya necesitas e-Kuatia.

Quien necesita e-Kuatia

Todo contribuyente que no cumpla con los requisitos de e-Kuatia'i debe usar e-Kuatia. Pero tambien hay contribuyentes que podrian calificar para e-Kuatia'i y aun asi eligen e-Kuatia por sus funcionalidades superiores. Estos son los perfiles tipicos:

  • Negocios con multiples establecimientos: dos o mas sucursales, o un local con varios puntos de emision.
  • Volumen medio o alto de facturacion: cualquier negocio que necesite emitir un volumen significativo de documentos por mes.
  • Necesidad de integracion: empresas que quieren que su sistema contable, ERP o software de gestion emita DTE automaticamente, sin intervencion manual.
  • Tipos de documento avanzados: notas de credito electronicas, notas de debito, notas de remision, autofacturas.
  • Proveedores del Estado: la Resolucion DNIT 41/2025 exige DTE para todas las operaciones con entidades publicas. Un software certificado e-Kuatia da mas control y trazabilidad para estos requisitos.

Cuando migrar de e-Kuatia'i a e-Kuatia

Si empezaste con e-Kuatia'i y tu negocio crece, eventualmente vas a necesitar migrar. Estos son los indicadores claros de que llego el momento:

1

Abriste una segunda sucursal o punto de venta

e-Kuatia'i solo permite un establecimiento. Si abriste otro local, la migracion es obligatoria. No hay forma de operar dos puntos de emision desde e-Kuatia'i.

2

Necesitas emitir notas de credito electronicas

Si un cliente devuelve un producto o necesitas anular parcialmente una factura, la nota de credito electronica es el documento correcto. e-Kuatia'i no la soporta. Sin nota de credito, no podes ajustar el IVA correctamente.

3

Tu volumen de facturacion supera los limites

La DNIT monitorea el uso de e-Kuatia'i. Si tus parametros de facturacion superan los limites establecidos, la DNIT te va a notificar que debes migrar a e-Kuatia. No esperes la notificacion: si ves que tu volumen crece consistentemente, planifica la migracion con tiempo.

4

Queres automatizar la emision

Si seguis emitiendo facturas una por una desde el portal web de la DNIT y eso te consume horas por semana, un software e-Kuatia con integracion API puede automatizar gran parte del proceso. El portal web es suficiente para 10 facturas al mes. No para 100.

Importante: la migracion de e-Kuatia'i a e-Kuatia no es automatica. Requiere contratar un software certificado, obtener una nueva habilitacion de la DNIT, y configurar la integracion tecnica. Planifica al menos 2 a 4 semanas para el proceso completo. Para saber si tu grupo SIFEN ya esta obligado, consulta como verificar tu grupo.

Lo que tu contador necesita saber

Si sos contador y manejas clientes que usan ambos sistemas, hay diferencias practicas que impactan tu trabajo diario:

Los documentos se ven igual

Para el receptor, un DTE emitido desde e-Kuatia'i y uno emitido desde e-Kuatia son identicos. Ambos son XML validados por el SIFEN con el mismo formato, el mismo CDC, y el mismo QR verificable. La diferencia esta en el emisor, no en el documento.

El procesamiento es el mismo

Cuando recibis facturas de los clientes de tu estudio, no importa desde cual plataforma se emitieron. Un KuDE es un KuDE. Los datos fiscales que necesitas para la liquidacion de IVA o IRP estan en el mismo lugar, independientemente del sistema de origen. Si queres conocer los desafios especificos para estudios contables, tenemos una guia dedicada.

La mezcla de formatos persiste

Algunos de tus clientes emiten con e-Kuatia'i, otros con e-Kuatia, y otros todavia usan factura preimpresa o virtual. Tu flujo de trabajo tiene que manejar los tres formatos. Errores frecuentes al elegir sistema

1

Elegir e-Kuatia'i para "ahorrar" cuando el negocio ya lo supero

e-Kuatia'i es gratuito, pero si tu negocio supera sus limites y seguis usandolo, estas emitiendo documentos en un sistema que no corresponde a tu perfil. Eso puede generar problemas ante una fiscalizacion. El ahorro de $15/mes no justifica el riesgo.

2

Contratar e-Kuatia cuando e-Kuatia'i es suficiente

El error inverso. Si tenes un solo local, bajo volumen y solo emitis facturas, no necesitas pagar un software. e-Kuatia'i cumple perfectamente. No inviertas en funciones que no vas a usar.

3

No verificar si el software esta certificado

Para usar e-Kuatia, necesitas un software homologado por la DNIT. No cualquier programa que "genere XML" es valido. Verifica en la lista oficial de la DNIT que el proveedor este certificado antes de contratar.

4

Confundir emision con recepcion

Tanto e-Kuatia como e-Kuatia'i son sistemas de emision. Si lo que necesitas es recibir y procesar las facturas electronicas que tus proveedores emiten, ninguno de los dos resuelve ese problema. Para eso necesitas una herramienta de procesamiento de facturas recibidas.

Preguntas frecuentes

Cual es la diferencia principal entre e-Kuatia y e-Kuatia'i?
e-Kuatia'i es la plataforma gratuita de la DNIT para pequenos contribuyentes con un solo establecimiento y bajo volumen. e-Kuatia es el sistema completo que permite integracion via API con software de terceros, multiples establecimientos y volumen ilimitado.
Puedo usar e-Kuatia'i si tengo mas de una sucursal?
No. e-Kuatia'i esta limitado a contribuyentes con un unico establecimiento y un unico punto de emision. Si tenes mas de una sucursal, necesitas usar e-Kuatia con un software certificado.
e-Kuatia'i es gratis?
Si. e-Kuatia'i es una herramienta gratuita proporcionada por la DNIT. No tiene costo de licencia ni suscripcion. Sin embargo, sus funcionalidades son limitadas comparadas con las soluciones e-Kuatia.
Que pasa cuando mi negocio crece y supero los limites de e-Kuatia'i?
Debes migrar a e-Kuatia con un software certificado. La DNIT notifica cuando un contribuyente supera los parametros. La transicion requiere contratar un software homologado, obtener la habilitacion y configurar la integracion via API con el SIFEN. Planifica al menos 2 a 4 semanas.
Un estudio contable puede usar e-Kuatia'i para sus clientes?
e-Kuatia'i es para que el propio contribuyente emita sus facturas. Un estudio contable que procesa facturas de multiples clientes necesita herramientas de recepcion y procesamiento como Fakturas.app, que trabaja con documentos emitidos desde cualquiera de los dos sistemas.
